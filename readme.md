## To get started
1. Clone the repo:
`git clone https://github.com/yashpriyam/movie-zone-project.git`

2. Get inside the cloned directory:
`cd movie-zone-project`

3. Install client dependencies:
`npm i`

4. Start client app:
`npm run app`

4. Change to server dir:
`cd server`

5. Install server dependencies:
`npm i`

4. Start server app:
`npm run dev`
